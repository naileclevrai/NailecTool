## How to use?


# Get For Kali/Debian
- Update

```
$ apt update
```

- install updates

```
$ apt upgrade 
```

- install dependencies

```
$ apt install git php openssh curl wget -y
```

- Install Script

```
$ git clone https://github.com/NailecLeVrai/NailecTool 
```

- Go in Folder

```
$ cd NailecTool
```

- Make it executable:

```
$ chmod +x ./menu.sh
```

- Run it:

```
$ ./menu.sh
```
